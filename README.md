# Ficha de RPG Web
Projeto de site para a criação de fichas do sistema de RPG "DaggerHeart".

Você pode utilizá-lo em:
https://larryfsg.github.io/Ficha-DaggerHeart/


## Objetivo

Facilitar o uso do sistema DaggerHeart com uma ficha digital,
rápida, acessível e interativa, sem login ou instalação.


## Como usar
- Acesse o site pelo link acima
- Preencha os campos da ficha
- Utilize os botões de cada um dos atributos para realizar as rolagens
- Marque as caixas de vantagem ou desvantagem para as rolagens
- Utilize os botões de rolar em cada uma das armas para rolar seus danos
- Utilize os botões nas armas do inventário para equipá-las


## Funcionalidades 
- Rolagem de dados de atributos
- Rolagem de dados de ataque
- Sistema de vantagem e desvantagem
- Troca de armas automática
- Importação e exportação da ficha

### Importação e Exportação
- Exporta a ficha em formato JSON
- Importa a ficha mantendo todos os dados salvos
- **IMPORTANTE: Não se esqueça de exportar a ficha sempre que fechar o site**

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)
